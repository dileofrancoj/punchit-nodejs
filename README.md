### Consigna TP FINAL

<p>Para el proyecto final se debe armar una aplicacion web que cumpla las siguientes condiciones</p>

- Tener al menos 4 rutas
- Implementar al menos 2 middlewares
- Implementar JWT
- Dockerizar el proyecto
- Deploy (AWS / Heroku) nginx
